# ESP32 Handheld Game Console

A fully functional handheld gaming console built on the ESP32, featuring 5 playable games.

## Games
- Dino Run
- Snake
- Tetris
- TimeTap

## Features
- State-machine-based game logic for smooth transitions
- Joystick-based navigation
- Real-time graphics on an ST7735 TFT display (SPI)

## Tech Stack
- **MCU:** ESP32
- **Display:** ST7735 TFT (SPI)
- **Input:** Joystick module
- **Language:** Embedded C
- **IDE:** Arduino IDE

## About
Built to explore game logic, display drivers, and real-time input handling on a microcontroller.
